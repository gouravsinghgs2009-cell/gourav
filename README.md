# Happy Birthday Project

A birthday surprise web page created with HTML, CSS, and JavaScript.

## Structure
- `index.html` — main page file
- `assets/images/` — memory images used in the gallery
- `assets/audio/` — birthday music audio file
- `.gitignore` — common files to ignore for Git

## GitHub Pages
1. Create a GitHub repository.
2. Copy these files into the repository root.
3. Commit and push.
4. In GitHub repo settings, enable GitHub Pages from the `main` branch and `/` root folder.

## Setup
- Add your images to `assets/images/`.
- Add your audio file to `assets/audio/birthday-music.mp3`.
- Do not use absolute local paths (`C:\` or `E:\`) in the published version.
